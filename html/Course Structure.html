Course Structure

This course is divided into 2 units. Each unit is aimed towards 6 weeks.

Unit 1

Content

Week Category Concept Java Specific Topics
1 Introduction Compilers JDK, JRE
2 Basics Variables Integers, Strings, Booleans
3 Basics Operators ++, --, *, /, %, +, -
4 Basics Expressions Operations and evaluations, Arrys
5 Basics Decision Statements If/Else/Else If, Switch Statements
6 Basics Loops For, While, Do While

Content

Week Category Concept Java Specific Topics

2 Objects How to Create Constructors
